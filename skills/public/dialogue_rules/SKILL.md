# Dialogue Writing Rules

## Core Principles
1. **Show, Don't Tell**: Characters should reveal intent through subtext, not blatant exposition.
2. **Distinct Voices**: Every character must have a unique vocabulary and speech cadence.
3. **Pacing**: Short, snappy dialogue drives tension; longer monologues should be earned and rare.

## Format
- Character names must be ALL CAPS and centered (or clearly delineated).
- Parentheticals (beat), (sighs) must be on a separate line below the character name.
- Do not overuse parentheticals; let the dialogue imply the action when possible.
