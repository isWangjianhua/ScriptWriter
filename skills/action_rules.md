# Action and Combat Protocol

## Visceral Description
1. Action paragraphs must be lean and muscular. Avoid excessive adjectives. Use strong verbs.
2. Break action lines into short chunks (2-3 sentences max) to control the reader's pace and mimic camera shots.
3. ALL CAPS should only be used to highlight key props, sounds, or the *first* introduction of a character.

## Spatial Awareness
1. Always establish the geography of the fight. Where are the characters relative to the exits, weapons, or hazards?
2. Actions must have consequences. If someone is hit in the leg, they must act with a limp in subsequent scenes.
