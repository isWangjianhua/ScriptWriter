# Hollywood Script Formatting

1. **Scene Headings (Sluglines)**: Must start with INT. or EXT., followed by LOCATION, followed by TIME (DAY, NIGHT, CONTINUOUS). All caps.
2. **Action lines**: Standard prose, present tense.
3. **Transitions**: (e.g., CUT TO:, FADE OUT.) are right-justified. Use them sparingly.
4. Voiceovers are marked as (V.O.) and OFF SCREEN dialogue as (O.S.) next to the character's name.
